Summary: Latitude vs. X

This project analyzes how weather changes at different distances from the equator.

To perform this analysis, we pulled data from the OpenWeatherMap API in order to create a dataset of over 500 cities.

Matplotlib was used to plot various aspects of the weather (temp, cloudiness, humidity, and windspeed) against the latitude. This site shares the source data and visualizations created during the analysis, as well as an analysis of trends observed.
